欢迎来到 **流金梦坊**！这不仅是一个简单的生成器，它是专门为创作者打造的“灵感加速器”。

你是否曾为一个新角色的背景故事而抓耳挠腮？或者想看自己笔下的两个角色进行互动，却不知道如何落笔？本项目的目标就是通过 **DeepSeek** 等先进的大语言模型，结合精妙的提示词工程，帮你一键补全人设，甚至让角色在故事中“活”过来。

---

## 核心功能 (Features)

*   **混合模式 OC 生成：** 只需提供几个关键词，剩下的背景故事、性格细节、外貌描写，交由我们来为你润色。
*   **多角色互动故事：** 注入两个角色的完整设定，只需一个场景（如：“在雨中的咖啡馆偶遇”），即可生成一篇 500-1000 字的高质量短剧。
*   **拒绝 OOC (Out Of Character)：** 深度上下文管理，确保角色在故事中的一言一行都严格遵循你设定好的性格逻辑。
*   **结构化输出：** 生成的内容自动符合 Markdown/JSON 格式，完美对接 Web 应用，让保存和展示变得轻而易举。

---

## 如何运作？ (How it Works)

本工具的核心并非从零训练模型，而是通过**提示词工程 (Prompt Engineering)** 对成熟的 LLM 进行特化引导：

1.  **System Prompt 注入：** 我们让模型扮演一位“深谙二次元文化的专业小说家”。
2.  **上下文拼接：** 在故事生成中，我们会动态拼接角色 A、角色 B 的人设，以及用户输入的场景关键词，形成一个高优先级的指令环境。
3.  **API 驱动：** 通过标准化的 API 接口进行调用，易于集成到任何前端或后端服务中。

## 如何在本地运行 (How to Run)

如果你下载了本项目的 ZIP 包，按照以下步骤即可在自己的电脑上跑起来：

### 1. 环境准备
*   安装 [Node.js](https://nodejs.org/) (推荐 v16 或更高版本)。
*   你需要一个 **DeepSeek API Key**。如果没有，请前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 申请。

### 2. 解压与安装
1.  解压下载的 ZIP 文件。
2.  打开终端（Terminal 或 PowerShell），进入项目目录下的 `preview-bootstrap` 文件夹：
    ```bash
    cd preview-bootstrap
    ```
3.  安装必要的依赖库：
    ```bash
    npm install
    ```

### 3. 配置环境变量
1.  进入 `preview-bootstrap/backend` 目录。
2.  找到 `.env` 文件（如果没有，请新建一个`.env`文件）。
3.  使用记事本或编辑器打开 `.env`，修改以下关键项：
    *   `DEEPSEEK_KEY`: 填入你的 DeepSeek API Key。
    *   `DB_HOST/USER/PASS`: 如果你使用本地 MySQL，请修改为你的数据库信息（默认为开发环境配置）。

### 4. 启动程序
回到 `preview-bootstrap` 目录，运行：
```bash
npm run dev
```
程序启动后，终端会显示 `✅ [Server] 服务已启动: http://localhost:3000`。

### 5. 开始使用
打开浏览器，访问 `http://localhost:3000`，即可进入“流金梦坊”的交互界面。
